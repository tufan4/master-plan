# Changelog

## Features Implemented

1.  **Deep Indexing & Recursive Structure**
    -   Implemented `RecursiveTopicNode` to support infinite nesting levels (e.g. `1.1.1.1`).
    -   Updated `curriculum.json` to demonstrate deep hierarchy.

2.  **Smart Action Buttons**
    -   Added "Vertical Action Group" with **PDF**, **Academic**, **Application**, and **YouTube** buttons.
    -   Implemented staggered hover animations.

3.  **Dictionary Engine**
    -   Created `Dictionary.tsx` with A-Z sorting, grouping, and search.
    -   Integrated Dictionary into Sidebar and Main Layout.

4.  **Dynamic Topic Addition**
    -   Added "**+ Yeni Alan Ekle**" button.
    -   Implemented simulated Supabase integration (using LocalStorage).

5.  **UI/UX Enhancements**
    -   Emerald Green theme for completion.
    -   Slate Dark aesthetics.
    -   Framer Motion animations.
